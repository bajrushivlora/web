Swift UI + Xcode Work

Coding
Interaction

I had the opportunity to learn Apple's Swift UI and Xcode frameworks. Xcode is a program used to develop software for Apple's iOS interfaces, and SwiftUI is a user interface framework used to build out iOS interfaces.

In the semester-long course, I was able to create in-app interactions with native iOS components. At the end of the course, I successfully built out a sample app that involves user interactions for a simple game.
